Technology

Built With

🍓
Raspberry Pi Zero 2W
Quad-core ARM Cortex-A53 at 1GHz. Runs all inference locally — no Wi-Fi, no cloud, no latency penalty.
Edge Hardware
👁️
YOLOv8-tiny
Optimised tiny variant of YOLOv8 for real-time single-board inference. ONNX export with quantised INT8 for speed.
Object Detection
📐
OpenCV Stereo
StereoSGBM disparity mapping from synchronised dual cameras. Obstacle distance computed per bounding box centroid.
Depth Estimation
🔤
Tesseract OCR
Motion-triggered OCR pipeline. Pre-processing via adaptive thresholding boosts accuracy on printed and handwritten text.
Text Recognition
📳
GPIO Haptics
PWM-controlled vibration motors via GPIO pins. Intensity mapped to inverse distance — closer object = stronger pulse.
Tactile Feedback
🎧
BT Audio + pyttsx3
Open-ear Bluetooth glasses deliver speech output without blocking ambient sound. pyttsx3 handles offline TTS synthesis.
